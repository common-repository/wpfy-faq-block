Gutenberg Block plugin for Frequently Asked Questions (FAQ) feature. Very straight forward to use. Just install and enjoy. 

== Description ==

This is a very straightforward tiny but useful plugin to add a block element that will help to create the Frequently Asked Questions (FAQ) section. It is very easy to use. After installing this plugin, a custom block item will be available, which will have a repeater pair of fields to add a question and answer. In the front end, it will generate a nice and smooth collapsible FAQ section.

* No Extra library or script
* Very lightweight and easy to use

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from the Plugins page.
